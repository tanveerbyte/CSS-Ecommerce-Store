# CSS Ecommerce Store

A fully responsive eCommerce store website built using **HTML5** and **CSS3**. The project focuses on creating a modern online shopping interface with responsive layouts, product showcases, navigation components, and a mobile-friendly user experience.

## Overview

The CSS Ecommerce Store project demonstrates how a complete eCommerce landing page can be developed using primarily HTML and CSS, with minimal JavaScript for navigation interactions. The website is designed to provide a clean shopping experience across desktop and mobile devices while maintaining a simple and lightweight codebase.

The project serves as a practical example of responsive web design, layout structuring, and front-end development fundamentals.

## Architecture

### HTML (`index.html`)

The HTML file serves as the foundation of the application and contains:

* Navigation bar and sidebar menu
* Hero/banner section
* Product showcase sections
* Promotional content blocks
* Footer section
* Embedded image references from external sources
* A small JavaScript snippet for sidebar open/close functionality

### CSS (`style.css`)

The CSS file is responsible for:

* Responsive page layouts
* Product card styling
* Navigation and sidebar presentation
* Typography and spacing
* Color schemes and visual hierarchy
* Hover effects and interactive states
* Mobile and desktop responsiveness

### Navigation Interaction

A lightweight JavaScript implementation is included within the HTML document to:

* Open the mobile navigation sidebar
* Close the mobile navigation sidebar
* Improve usability on smaller screen sizes

### External Assets

Images are sourced from external providers and referenced directly through image URLs within the HTML document. This approach keeps the repository lightweight while allowing the interface to display rich visual content.
